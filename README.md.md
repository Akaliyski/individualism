## Project Overview

This repository reproduces all analyses and figures for  
**Individualism-Collectivism: Reconstructing Hofstede’s Dimension of Cultural Differences**  
Author: Plamen Akaliyski (Lingnan University)  
Email: akaliyski.plamen@gmail.com  
Created: 2025-05-29  
Last updated: 2025-06-02

**Workflows:**
1. `Replication_stata.do` (Stata): validation, data preparation, index construction, validity checks, predictive models, and exports  
2. `Figures_R.R` (R): data visualizations and figure exports

---

## Repository Structure

```
Replication_Files/
├── README.md
├── Replication_stata.do    ← Stata preprocessing & exports
├── Figures_R.R             ← R‐based figure generation
├── Data/
│   ├── Raw_data/           ← Original .dta/.xlsx inputs
│   ├── Processed_data/     ← Generated by Replication_stata.do

    └── Figures from R/     ← Where Figures_R.R saves PNG/PDFs
```

## 1. Stata Workflow

**Filename:** `Replication_stata.do`  
**Location:** Top‐level `Replication_Files/`

**Run in Stata:**
1. Change directory to your local copy of the repository. For example:
   ```stata
   cd "C:\path	o\Replication_Files"
   do Replication_stata.do
   ```
2. Before running, edit the global `path` at the top of `Replication_stata.do` so it matches your folder. For example:
   ```stata
   global path "C:\path	o\Replication_Files"
   ```

**What this does:**
- Reads raw data from `Data/Raw_data`
- Writes cleaned datasets to `Data/Processed_data`
- Exports tables (Word/Excel) to `Output/`
- Saves R-ready `.dta` and `.xlsx` files to `Data/R_data` (​note: some R files are manually extracted or reconstructed)

---

## 2. R Workflow

**Filename:** `Figures_R.R`  
**Location:** Top‐level `Replication_Files/`

**Run in R:**
```r
setwd("C:/path/to/Replication_Files")
source("Figures_R.R")
```
- Reads from `Data/R_data` (​both auto-exported and manually prepared data files)
- Generates all figures (Figures 2, S1–S8, 3–10, S12) and saves them into `Output/Figures from R/`

---

## 3. Folder Descriptions

- **Data/Raw_data/**  
  Contains all original survey, cultural-index, and national statistics source files (Stata `.dta`, Excel `.xlsx`, etc.).

- **Data/Processed_data/**  
  Cleaned and merged Stata datasets created by running `Replication_stata.do`.

- **Data/R_data/**  
  - **Auto-exported by Stata**: e.g., `Difference_from_Hofstede.dta`, `All individualism measures.dta`.  
  - **Manually extracted/reconstructed**: Some Excel/CSV files are assembled by hand (e.g., “Indicators for Individualism.xlsx”) for use in R plots.

- **Output/**  
  - Tables exported by Stata (e.g., `Table2.doc`, `TableS1.doc`, `Footnote10.doc`).  
  - **Figures from R/**: All PNG/PDF figures produced by `Figures_R.R`.

## 4. Step-by-Step Reproduction

1. **Clone/download** this repository, preserving its folder structure.  
2. **Install** required software:  
   - Stata 18+ (install `asdoc`, `paran` via `ssc install asdoc` and `ssc install paran`)  
   - R 4.2+ (install packages listed at the top of `Figures_R.R`)  

3. **Run Stata**:
   ```stata
   cd "C:\path	o\Replication_Files"
   do Replication_stata.do
   ```
   This populates `Data/Processed_data`, `Data/R_data`, and exports tables to `Output/`.

4. **Run R**:
   ```r
   setwd("C:/path/to/Replication_Files")
   source("Figures_R.R")
   ```
   This generates all figures into `Output/Figures from R/`.

5. **Verify** that the resulting tables and figures match those in the publication.
